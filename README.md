# dvoice

A telephone over [dCast](https://github.com/hyperbeam)

```
npm install -g dvoice
```

Needs sox installed.

On mac:

```sh
brew install sox
```

On linux:

```sh
sudo apt-get install sox
```

## Usage

On your computer

```sh
dvoice "your name and your friend"
```

On your friends

```sh
dvoice "your name and your friend"
```

Now your mic will be streamed to your friends audio and vice versa!
